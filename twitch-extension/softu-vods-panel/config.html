Softu VODs Panel

This extension panel does not require runtime configuration. It reads recent VODs from https://softu.one/data/vods.json and links to https://softu.one/#/youtube/:vodId.

Required Twitch Extension allowlisted domains (CSP / Asset Host allowlist):

  • https://softu.one (VOD data + destination links)
  • https://static-cdn.jtvnw.net (Twitch thumbnails)
  • https://i.ytimg.com (YouTube thumbnails)

Optional preview override for local testing: add ?siteUrl=... or ?dataUrl=... to panel.html.
